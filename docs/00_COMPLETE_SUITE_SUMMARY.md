## 🚀 Quick Start for Each Guide

### Guide 01: Install OpenCode
```bash
# Fastest method
npm install -g opencode-ai
opencode --version
```

### Guide 02: Use Serena (Already Built-in!)
```bash
# Just use OpenCode in a code project
cd /path/to/project
opencode
> Show me the structure of main.py
```

### Guide 03: Understand Architecture
```
Read the guide to understand:
- Serena = built-in (automatic)
- MCP = external integrations (configure)
```

### Guide 04: Add GitHub
```bash
npm install -g @modelcontextprotocol/server-github
export GITHUB_PERSONAL_ACCESS_TOKEN="ghp_your_token"
# Configure in opencode.json
```

### Guide 05: Add Atlassian ⭐ NEW!
```bash
npm install -g @modelcontextprotocol/server-atlassian
export ATLASSIAN_CLOUD_ID="yourcompany.atlassian.net"
export ATLASSIAN_API_TOKEN="ATATT_your_token"
export ATLASSIAN_USER_EMAIL="your.email@company.com"
# Configure in opencode.json
```
